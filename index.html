Our Mission

Upon conducting research at Purdue, our team realized it was time for a change! There is no centralized place to find clubs and events on campus. There is also a lack in recruiting for a lot of organizations. This is where we step in. We are building a web-platform that will apply machine learning to find the best organizations and events on campus based on your interests for you! With our high tech software not only will you be more social and involved, but cool features can help you find events and interests that you never imagined.

Join us in making big campuses, smaller!

Features

  • ML-Based Algorithm
  • Explore Random Events
  • Mentoriship Avalibility
  • Events Map
  • Real-Time Calendar

There are many key components to our web platform. The journey of flyr will begin by asking the user to create a profile. In the profile there will be basic questions asked to help train the data to perform a Machine Learning based Algorithm. The more data collected, the more Flyr will be able to do it's job. The reccomendations will occur directly to you so, you won't have to go out of your way to get active on campus. There will be an explore random events page so that you can take part in things of interest, or of things that you never thought you would be willing to try. A lot of people are scared to go to a callout or that first meeting, so Flyr is creating the first-ever mentorship program so that you know atleast one person before going to that meeting. There will be an events map so you can see what events are happening near you. There will aslo be a Calendar so you can keep track of how involved you will get on campus! These are the current features being implemented as we speak!
